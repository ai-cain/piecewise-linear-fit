# Screenshots

This page collects the main UI screenshots used across the documentation.

## CSV Import

The CSV page loads measured points, optionally reuses CSV headers as display/export names, and shows the editable imported table.

![CSV Import](img/01_csv_import.png)

## Manual Input: Range Mode

Range mode generates evenly spaced `X` values so the user can fill `Y` manually.

![Manual Input Range](img/02_01_manual_input_range.png)

## Manual Input: Custom Points

Custom mode lets the user create irregular point distributions, edit both coordinates, and sort by `X`.

![Manual Input Custom](img/02_01_manual_input_custom.png)

## Results: Fit Chart

The fit chart view can show the combined piecewise fit, measured points only, or fitted lines only.

![Results Chart](img/03_01_results_chart.png)

## Results: Global Residual

This view compares all points against a single reference line inspired by the notebook workflow.

![Global Residual](img/03_02_results_global_residual.png)

## Results: Segment Error

This chart shows per-segment residuals, the review band, and highlighted out-of-range points.

![Segment Error](img/03_03_results_segment_error.png)

## Results: Code Export

The copy-results view contains segment cards, target selection, the generated code preview, and the copy action.

![Code Export](img/03_04_code_copy.png)
